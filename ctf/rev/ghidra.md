## Install

```zsh
sudo apt install openjdk-21-jdk

curl -LO https://github.com/NationalSecurityAgency/ghidra/releases/download/Ghidra_11.2.1_build/ghidra_11.2.1_PUBLIC_20241105.zip

~/ghidra_11.2.1_PUBLIC/ghidraRun &
```
